# Verben (verbs) 1–100

<!--
Format — a verb's "triple" is its PRINCIPAL PARTS (everything else is computed from them):
  bold row      = infinitive + English + the parts: 3rd-sg (only if stem-changing) · Präteritum · Partizip II
                  the auxiliary lives in the Partizip (hat … = haben, ist … = sein)
  sentence row  = a plain present-tense example
  fingerprint   = only what's TRUE OF THIS VERB and breaks a default:
                    governs a case/preposition (warten auf + Akk), is reflexive (sich freuen),
                    separable, or an idiom. Shared machinery stays in the grammar notes.
  Note column   = class tag:  ◾ weak (regular)   ◆ strong   ◈ mixed   ⚠️ irregular
  Blank | | | separates entries.

| **machen · machte · hat gemacht**                | to do, make               | ◾ weak — the regular default                |

| **sprechen · spricht · sprach · hat gesprochen** | to speak                  | ◆ strong · stem-change e→i                  |

| Deutsch **sprechen**                             | to speak German           |                                             |
|                                                  |                           |                                             |
| **fahren · fährt · fuhr · ist gefahren**         | to drive, go (by vehicle) | ◆ strong · stem-change a→ä · **ist**        |

| **denken · dachte · hat gedacht**                | to think                  | ◈ mixed — vowel change + weak -t-           |
