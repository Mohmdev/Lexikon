# Verben (verbs) 1–100

<!--
Format — a verb's "triple" is its PRINCIPAL PARTS (everything else computes from them):
  parts cell    = the principal parts stacked on separate lines, all bold:
                  infinitive / (3rd-sg only if stem-changing) / Präteritum / Partizip II
                  the auxiliary lives in the Partizip (hat … = haben, ist … = sein)
  sentence row  = a plain present-tense example
  extra rows    = a fixed object / phrase worth seeing
  Anmerkung     = class tag (◾ weak · ◆ strong · ◈ mixed · ⚠️ irregular) + only what's TRUE OF THIS VERB:
                  governs a case/preposition (warten auf + Akk), reflexive (sich freuen), separable, an idiom.
  Entries are separated by the taller stacked parts cell — no blank rows.
-->
