# Dativpräpositionen (always Dativ)

*These always force the **dative** on the noun behind them.*

A **closed list**. The core eight:
**aus – außer – bei – mit – nach – seit – von – zu** (plus *gegenüber*).

| German | English | Note |
|---|---|---|
| **aus** | out of, from | |
| aus **dem** Haus | out of the house | also origin: aus Deutschland (from Germany) |
| | | |
| **außer** | except, besides | |
| außer mir | except me | |
| | | |
| **bei** | at (someone's place), near | |
| bei mir | at my place | also: bei der Arbeit (at work) |
| | | |
| **mit** | with, by | |
| mit **dem** Bus | by bus | means of transport → mit |
| | | |
| **nach** | after, to | |
| nach **der** Schule | after school | also "to" cities/countries: nach Berlin |
| | | |
| **seit** | since, for | |
| seit **einem** Jahr | for a year | ongoing time up to now |
| | | |
| **von** | from, of | |
| von dir | from you | also possession: das Auto von meinem Bruder |
| | | |
| **zu** | to | |
| **zum** Arzt | to the doctor | zum = zu dem (people/places you go *to*) |
| | | |
| **gegenüber** | opposite, across from | usually *follows* a pronoun: mir gegenüber |
| gegenüber **dem** Park | opposite the park | |
