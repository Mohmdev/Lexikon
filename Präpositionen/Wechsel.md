## The rule: movement vs. location

- **movement / direction** — answers *wohin?* (where to) → **Akkusativ**
- **location / position** — answers *wo?* (where) → **Dativ**

| German | English | Note |
|---|---|---|
| Ich gehe **in die** Schule. | I go **into** the school. | Akkusativ — movement *(wohin?)* |
| Ich bin **in der** Schule. | I am **in** the school. | Dativ — location *(wo?)* |
| | | |
| Ich hänge das Bild **an die** Wand. | I hang the picture **onto** the wall. | Akkusativ — movement |
| Das Bild hängt **an der** Wand. | The picture hangs **on** the wall. | Dativ — location |

The feel of it: **moving into a new place → Akkusativ; already sitting there → Dativ.**
