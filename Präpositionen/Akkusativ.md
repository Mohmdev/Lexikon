# Akkusativpräpositionen (always Akkusativ)

*These always force the **accusative** on the noun behind them.*

| German | English | Note |
|---|---|---|
| **durch** | through | |
| durch **den** Park | through the park | |
| | | |
| **für** | for | |
| für dich | for you | |
| | | |
| **gegen** | against | |
| gegen **die** Wand | against the wall | |
| | | |
| **ohne** | without | |
| ohne mich | without me | |
| | | |
| **um** | around | |
| um **den** Tisch | around the table | |
| | | |
| **bis** | until, up to | |
| bis nächste Woche | until next week | |
| | | |
| **entlang** | along | follows the noun, not before it |
| **die** Straße entlang | along the street | |
