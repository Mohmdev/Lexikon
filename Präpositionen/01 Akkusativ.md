# Akkusativpräpositionen (always Akkusativ)

*These always force the **accusative** on the noun behind them — no exceptions, no thinking required.*

This is a **closed list**: grind the membership once. The core five chant as **durch – für – gegen –
ohne – um**; *bis* and *entlang* round it out.

| German | English | Note |
|---|---|---|
| **durch** | through | |
| durch **den** Park | through the park | |
| | | |
| **für** | for | |
| für dich | for you | |
| | | |
| **gegen** | against | |
| gegen **die** Wand | against the wall | also "around" a time: gegen 8 Uhr |
| | | |
| **ohne** | without | |
| ohne mich | without me | often used with no article: ohne Auto |
| | | |
| **um** | around | |
| um **den** Tisch | around the table | also clock time: um 8 Uhr (at 8) |
| | | |
| **bis** | until, up to | |
| bis nächste Woche | until next week | usually pairs up: *bis zu* + Dativ |
| | | |
| **entlang** | along | follows the noun, not before it |
| **die** Straße entlang | along the street | |
