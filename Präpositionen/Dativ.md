# Dativpräpositionen (always Dativ)

*These always force the **dative** on the noun behind them.*

| German | English | Note |
|---|---|---|
| **aus** | out of, from | |
| aus **dem** Haus | out of the house | |
| | | |
| **bei** | at (someone's place), near | |
| bei mir | at my place | |
| | | |
| **mit** | with, by | |
| mit **dem** Bus | by bus | |
| | | |
| **nach** | after, to | |
| nach **der** Schule | after school | |
| | | |
| **seit** | since, for | |
| seit **einem** Jahr | for a year | |
| | | |
| **von** | from, of | |
| von dir | from you | |
| | | |
| **zu** | to | |
| **zum** Arzt | to the doctor | zum = zu dem |
| | | |
| **gegenüber** | opposite, across from | |
| gegenüber **dem** Park | opposite the park | |
