# Fragewörter / W-Wörter (Question words)

<!-- Closed set. Almost all start with W. English on the right. -->

| German | English | Note |
|---|---|---|
| wer | who | the subject "who" |
| wen | whom | accusative of *wer* |
| wem | (to) whom | dative of *wer* |
| wessen | whose | genitive of *wer* |
| was | what | |
| wann | when | |
| wo | where | location — where something *is* |
| wohin | where to | direction — going somewhere |
| woher | where from | origin — coming from |
| warum | why | also: *wieso*, *weshalb* |
| wie | how | |
| wie viel | how much | |
| wie viele | how many | |
| welcher | which | takes article endings: welcher / welche / welches |

> **Woher** kommst du? — Where are you from?
> **Welches** Auto ist deins? — Which car is yours?
