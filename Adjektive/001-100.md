# Adjektive (adjectives) 1–100

<!--
Format:
  bold row       = the adjective + its English meaning
  comparison row = comparative / superlative (regular = +er / am ...sten)
  sentence row   = a real sentence (predicative: after "ist")
  ↔ row          = its opposite (natural pair, study them together)
  ⚠️ on the bold row = comparison is IRREGULAR — can't guess it, memorize it.
  Blank | | | | separates entries.
-->

| German | English | Note |
|---|---|---|
| **schnell** | fast | |
| schneller / am schnellsten | faster / fastest | |
| Das Auto ist **schnell**. | The car is fast. | |
| ↔ langsam | slow | |
| | | |
| **langsam** | slow | |
| langsamer / am langsamsten | slower / slowest | |
| Der Zug ist **langsam**. | The train is slow. | |
| ↔ schnell | fast | |
| | | |
| **gut** | good | ⚠️ irregular comparison |
| besser / am besten | better / best | memorize — not "guter/gutest" |
| Das Essen ist **gut**. | The food is good. | |
| ↔ schlecht | bad | |
| | | |
| **schlecht** | bad | |
| schlechter / am schlechtesten | worse / worst | -t stem → superlative adds **-est** |
| Das Wetter ist **schlecht**. | The weather is bad. | |
| ↔ gut | good | |
| | | |
| **groß** | big, tall | ⚠️ umlaut in comparison |
| größer / am größten | bigger / biggest | memorize the ö |
| Der Mann ist **groß**. | The man is tall. | |
| ↔ klein | small | |
| | | |
| **klein** | small | |
| kleiner / am kleinsten | smaller / smallest | |
| Die Katze ist **klein**. | The cat is small. | |
| ↔ groß | big | |
| | | |
| **hoch** | high | ⚠️ drops the **c** before endings |
| höher / am höchsten | higher / highest | comparison also irregular |
| ein **hohes** Haus | a high house | before a noun it's **hoh-**, not "hoch-" |
| Der Berg ist **hoch**. | The mountain is high. | |
