# Nomen (nouns) 1–100

<!--
Format — a noun's "triple" is article + singular + PLURAL (never learn a noun naked):
  bold row     = the triple (der/die/das + singular, die + plural) — learn all three together
  sentence row = a plain example sentence
  fingerprint  = only what's TRUE OF THIS NOUN and breaks a default:
                   an odd plural, a surprising 2nd sense, an n-noun ending, a fixed idiom.
                   Shared machinery (cases, prepositions) stays in the grammar notes.
  Note column  = gender tag:  🔵 der (masc)   🔴 die (fem)   🟢 das (neut)
  Blank | | | separates entries.
-->
