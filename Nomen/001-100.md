# Nomen (nouns) 1–100

<!--
Format:
  bold row     = the triple (der/die/das + singular + plural) — always learn all three together
  rows under it = usage: a plain example sentence first, then any fingerprints (odd senses, idioms)
  Note column  = tag only. Shared machinery (am/auf/unter Tisch…) stays in the grammar notes, not here.

| German | English | Note |
|---|---|---|
| **der Tisch, die Tische** | table | 🔵 |
| Der Tisch ist groß. | The table is big. | |
| den Tisch decken | to set the table | verb it pairs with |
| zu Tisch! | dinner's ready! | idiom, breaks pattern |
| | | |
| **die Frau, die Frauen** | woman | 🔴 |
| Die Frau liest. | The woman is reading. | |
| meine Frau | my wife | surprising 2nd sense |
| Frau Müller | Mrs. Müller | also = "Mrs." |
| | | |
| **das Haus, die Häuser** | house | 🟢 |
| Das Haus ist alt. | The house is old. | |
| nach Hause | (going) home | fixed form, odd -e |
| zu Hause | at home | fixed form |
