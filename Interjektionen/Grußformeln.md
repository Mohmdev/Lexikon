# Grußformeln (greetings & social formulas)

*Fixed social phrases — hellos, goodbyes, thanks, please. You don't build these from grammar, you just
**know** them, so they live here as a list. Most have a formal and a casual version.*

| German | English | Note |
|---|---|---|
| **hallo** | hello | all-purpose, casual |
| **Hallo**, wie geht's? | Hello, how are you? | |
| | | |
| **hi** | hi | very casual |
| **Hi**, alles klar? | Hi, all good? | |
| | | |
| **hey** | hey | casual, also to get attention |
| **Hey**, warte mal! | Hey, wait a sec! | |
| | | |
| **moin** | hi / morning | northern Germany, any time of day |
| **Moin**, na? | Hi, how's it going? | |
| | | |
| **servus** | hi / bye | southern Germany & Austria, works both ways |
| **Servus**, bis später! | Hi / bye, see you later! | |
| | | |
| **guten Tag** | good day | formal, daytime |
| **Guten Tag**, Herr Müller. | Good day, Mr Müller. | |
| | | |
| **guten Morgen** | good morning | until ~11 a.m. |
| **Guten Morgen!** | Good morning! | |
| | | |
| **tschüss** | bye | casual goodbye |
| **Tschüss**, bis morgen! | Bye, see you tomorrow! | |
| | | |
| **auf Wiedersehen** | goodbye | formal goodbye |
| **Auf Wiedersehen!** | Goodbye! | lit. "until seeing again" |
| | | |
| **bis bald** | see you soon | |
| Mach's gut, **bis bald!** | Take care, see you soon! | |
| | | |
| **danke** | thanks | |
| **Danke** für die Hilfe. | Thanks for the help. | |
| | | |
| **bitte** | please / you're welcome | ⚠️ many jobs (see note below) |
| **Bitte** schön. — „Danke." — **Bitte.** | Here you go. — "Thanks." — You're welcome. | |
| | | |
| **entschuldigung** | sorry / excuse me | apology *and* getting attention |
| **Entschuldigung**, wo ist der Bahnhof? | Excuse me, where's the station? | |
| | | |
| **gern / gerne** | gladly | warm reply to thanks |
| „Danke!" — **Gern!** | "Thanks!" — My pleasure! | |

> ⚠️ **bitte** is a multi-tool: "please", "you're welcome", "here you go", and "pardon?" — the
> situation tells you which.
