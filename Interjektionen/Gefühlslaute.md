# Gefühlslaute (emotion sounds)

*Sounds expressing a feeling — surprise, pain, disgust, relief — plus common conversation fillers.
They sit **outside** the sentence (set off by a comma or an exclamation mark) and never change form.*

| German | English | Note |
|---|---|---|
| **ach** | oh, ah | realization or mild dismay |
| **Ach**, jetzt verstehe ich. | Oh, now I get it. | |
| | | |
| **oh** | oh | surprise |
| **Oh**, das ist schön! | Oh, that's nice! | |
| | | |
| **oje / o je** | oh dear | worry |
| **Oje**, ich habe es vergessen. | Oh dear, I forgot it. | |
| | | |
| **aua / au** | ouch | pain |
| **Aua!** Das tut weh. | Ouch! That hurts. | |
| | | |
| **igitt** | yuck, ew | disgust |
| **Igitt**, das schmeckt schlecht! | Yuck, that tastes bad! | |
| | | |
| **pfui** | shame, ugh | disapproval — old-fashioned / storybook |
| **Pfui**, so etwas sagt man nicht. | Ugh, you don't say things like that. | |
| | | |
| **hurra** | hooray | joy — old-fashioned / storybook |
| **Hurra**, wir haben gewonnen! | Hooray, we won! | |
| | | |
| **mensch** | man!, geez! | mild exasperation or surprise |
| **Mensch**, wo warst du? | Man, where were you? | |
| | | |
| **wow** | wow | amazement |
| **Wow**, das ist groß! | Wow, that's big! | |
| | | |
| **boah** | whoa | awe / overwhelm |
| **Boah**, ist das heiß! | Whoa, it's hot! | |
| | | |
| **na** | well… | prompting or hesitation |
| **Na**, wie war's? | Well, how was it? | |
| | | |
| **na ja** | well, sort of | resignation |
| **Na ja**, es geht so. | Well, it's so-so. | |
| | | |
| **tja** | well, hmm | resigned shrug |
| **Tja**, da kann man nichts machen. | Well, nothing you can do. | |
| | | |
| **hm** | hmm | thinking or doubt |
| **Hm**, ich bin nicht sicher. | Hmm, I'm not sure. | |
| | | |
| **äh / ähm** | uh, um | hesitation filler — the most common one |
| Ich, **äh**, weiß nicht. | I, uh, don't know. | |
| | | |
| **hä** | huh? | confusion / didn't catch it |
| **Hä?** Das verstehe ich nicht. | Huh? I don't get that. | |
| | | |
| **ne? / oder?** | right?, isn't it? | tag tacked on to seek agreement |
| Das ist gut, **ne?** | That's good, right? | |
| | | |
| **puh** | phew | relief or effort |
| **Puh**, das war knapp! | Phew, that was close! | |
