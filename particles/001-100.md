# Partikeln (Particles) — the flavor words

<!--
Closed set. These add TONE / attitude, and most have NO clean English translation — so the middle
column is their FUNCTION, not a word-for-word gloss. You absorb these by ear as much as by table.
-->

| German | what it does | example |
|---|---|---|
| doch | contradicts a negative — "but yes it is!" | Komm **doch**! — Oh come on, do come! |
| mal | softens a request — "just" | Sag **mal**… — Say, … / Tell me… |
| ja | "as you know" / mild surprise | Das ist **ja** toll! — Well, that's great! |
| halt / eben | resignation — "that's just how it is" | Es ist **halt** so. — It just is what it is. |
| denn | warmth in a question — "so / then" | Was machst du **denn**? — So what are you up to? |
| schon | reassurance — "it'll be fine" | Das wird **schon**. — It'll work out. |
| wohl | "probably / I guess" | Er ist **wohl** krank. — He's probably ill. |

These carry tone, not content — don't expect a tidy English equal for each.
