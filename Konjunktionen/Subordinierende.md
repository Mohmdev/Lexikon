# Subordinierende Konjunktionen (Subordinating)

*Open a dependent clause — the conjugated verb jumps to the END.*

| German | English | Note |
|---|---|---|
| **weil** | because | the everyday "because" |
| Ich bleibe, **weil** ich müde **bin**. | I'm staying because I'm tired. | |
| | | |
| **da** | since, because | reason already known; more formal |
| **Da** es **regnet**, bleiben wir. | Since it's raining, we're staying. | |
| | | |
| **dass** | that | |
| Ich weiß, **dass** er **kommt**. | I know that he's coming. | |
| | | |
| **ob** | whether | |
| Ich frage, **ob** er **kommt**. | I'm asking whether he's coming. | |
| | | |
| **wenn** | if, when | a repeating or future situation |
| **Wenn** es **regnet**, bleibe ich. | If it rains, I stay. | |
| | | |
| **als** | when | a single event in the past |
| **Als** ich klein **war**, … | When I was little, … | |
| | | |
| **während** | while, whereas | |
| **Während** sie **kocht**, lese ich. | While she cooks, I read. | |
| | | |
| **bevor** | before | |
| **Bevor** ich **gehe**, esse ich. | Before I leave, I eat. | |
| | | |
| **ehe** | before | formal / written |
| **Ehe** du **gehst**, ruf mich an. | Before you go, call me. | |
| | | |
| **nachdem** | after | |
| **Nachdem** wir **gegessen haben**, gehen wir. | After we've eaten, we go. | |
| | | |
| **seit / seitdem** | since | a point in time |
| **Seit** ich hier **wohne**, bin ich glücklich. | Since I've lived here, I'm happy. | |
| | | |
| **bis** | until | |
| Warte, **bis** ich **komme**. | Wait until I come. | |
| | | |
| **sobald** | as soon as | |
| **Sobald** er **kommt**, essen wir. | As soon as he comes, we eat. | |
| | | |
| **solange** | as long as | |
| **Solange** es **regnet**, bleibe ich. | As long as it rains, I stay. | |
| | | |
| **obwohl** | although | |
| Ich gehe, **obwohl** ich müde **bin**. | I'm going although I'm tired. | |
| | | |
| **obgleich** | although | formal |
| Ich bleibe, **obgleich** es spät **ist**. | I'm staying although it's late. | |
| | | |
| **damit** | so that | purpose |
| Ich lerne, **damit** ich die Prüfung **bestehe**. | I study so that I pass the exam. | |
| | | |
| **sodass** | so that | result |
| Es regnet, **sodass** wir zu Hause **bleiben**. | It's raining, so that we stay home. | |
| | | |
| **falls** | in case, if | |
| **Falls** es **regnet**, nimm den Schirm. | In case it rains, take the umbrella. | |
| | | |
| **sofern** | provided that | formal |
| Ich helfe, **sofern** ich **kann**. | I'll help, provided I can. | |
| | | |
| **indem** | by (doing) | means / method |
| Er lernt, **indem** er viel **liest**. | He learns by reading a lot. | |
| | | |
| **ohne dass** | without (…ing) | |
| Er ging, **ohne dass** ich es **merkte**. | He left without my noticing. | |
| | | |
| **anstatt dass** | instead of (…ing) | |
| **Anstatt dass** er **hilft**, schläft er. | Instead of helping, he sleeps. | |
| | | |
| **als ob** | as if | |
| Er tut, **als ob** er **schläft**. | He acts as if he's asleep. | |
| | | |
| **wie** | as, like | manner |
| Mach es, **wie** ich es **mache**. | Do it the way I do it. | |
