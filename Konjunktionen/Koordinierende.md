# Koordinierende Konjunktionen (Coordinating)

*Join two equal parts — no change to word order.*

| German | English | Note |
|---|---|---|
| **und** | and | |
| Ich lese **und** sie schläft. | I'm reading and she's sleeping. | |
| | | |
| **oder** | or | |
| Trinkst du Tee **oder** Kaffee? | Are you drinking tea or coffee? | |
| | | |
| **aber** | but | |
| Ich bin müde, **aber** ich bleibe. | I'm tired, but I'm staying. | |
| | | |
| **denn** | because, for | gives a reason — but **no** verb flip (unlike *weil*) |
| Ich bleibe, **denn** ich bin müde. | I'm staying, because I'm tired. | |
| | | |
| **sondern** | but rather | only after a negative |
| Das ist nicht rot, **sondern** blau. | That's not red, but blue. | |
| | | |
| **sowie** | as well as | |
| Wir haben Hunde **sowie** Katzen. | We have dogs as well as cats. | |
| | | |
| **doch / jedoch** | however | *jedoch* can also start a clause and flip the verb |
| Er ist klein, **doch** er ist stark. | He's small, but he's strong. | |
