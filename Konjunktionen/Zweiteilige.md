# Zweiteilige Konjunktionen (Two-part)

*A fixed pair that wraps two things.*

| German | English | Note |
|---|---|---|
| **entweder … oder** | either … or | |
| Ich trinke **entweder** Tee **oder** Kaffee. | I drink either tea or coffee. | |
| | | |
| **weder … noch** | neither … nor | |
| **Weder** er **noch** ich war da. | Neither he nor I was there. | |
| | | |
| **sowohl … als auch** | both … and | |
| Ich mag **sowohl** Tee **als auch** Kaffee. | I like both tea and coffee. | |
| | | |
| **nicht nur … sondern auch** | not only … but also | |
| Er ist **nicht nur** klug, **sondern auch** nett. | He's not only smart but also nice. | |
| | | |
| **zwar … aber** | admittedly … but | |
| Es ist **zwar** klein, **aber** schön. | It's small, admittedly, but nice. | |
| | | |
| **je … desto / umso** | the … the | with comparatives |
| **Je** mehr, **desto** besser. | The more, the better. | |
