# Zweiteilige Konjunktionen — paired connectors

These connectors form a **fixed two-part frame** around both halves of what is joined:
*either… or*, *neither… nor*, *not only… but also*. The first word opens the frame, the second
closes it. They are a small **closed set**; learn the pairs as units.

---

## The common pairs

| German pair | English | example |
|---|---|---|
| **entweder … oder** | either … or | Ich nehme **entweder** Tee **oder** Kaffee. |
| **weder … noch** | neither … nor | **Weder** er **noch** ich war da. |
| **sowohl … als auch** | both … and | Ich mag **sowohl** Tee **als auch** Kaffee. |
| **nicht nur … sondern auch** | not only … but also | Er ist **nicht nur** klug, **sondern auch** nett. |
| **zwar … aber** | admittedly … but | Es ist **zwar** klein, **aber** schön. |
| **je … desto / umso** | the … the (+ comparatives) | **Je** mehr, **desto** besser. |

---

## Common errors

### weder … noch already contains the negation

It means **neither/nor**, so the sentence is **already negative**; do not add another *nicht/kein*:

> **Weder** er **noch** sie **kommt**. — Neither he nor she is coming. *(no extra "not")*

(With *weder…noch* the verb often stays singular when each item is singular, as above.)

### je … desto — word order

This is the only pair with its **own** grammar. Both halves take a
**comparative** ([[05 Steigerung]]), and the two clauses behave differently:

> **Je** mehr ich **lerne**, **desto** besser **verstehe** ich.
> — The more I learn, the better I understand.

- **je**-clause = subordinate → **verb to the END** (*…lerne*).
- **desto**-clause = main → starts with *desto + comparative*, then **verb in 2nd place** (*…verstehe*).

So the shape is: **Je** [comparative] [subject] [**verb-end**] **, desto** [comparative] [**verb**]
[subject]. *(umso is a free swap for desto.)*

---

## Where they sit grammatically

Most of these pairs link **equal** elements, so they don't disturb the verb (like coordinating
connectors — [[02 Koordinierende]]). The exception is **je … desto**, which mixes a subordinate clause
with a main one as shown above.

---

## Summary

Six pairs, learned as fixed units. The only one with special grammar is **je … desto** (verb-final
*je*-clause, then *desto* + verb-second). The rest frame two equal parts. → [[01 Wortstellung]]
