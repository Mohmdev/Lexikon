# Gradadverbien (degree adverbs)

*🔆 Answer **wie sehr?** (how much).*

These don't take a sentence slot; they **cling to an adjective or adverb** and crank its intensity.

| German | English | Note |
|---|---|---|
| **sehr** | very | |
| Es ist sehr kalt. | It is very cold. | intensifies adj/adv only; verbs use "viel" (ich arbeite viel) |
| | | |
| **zu** | too (excessively) | |
| Das ist zu teuer. | That's too expensive. | careful: "zu" is also the preposition "to" |
| | | |
| **ganz** | quite / completely | |
| ganz gut / ganz kaputt | quite good / completely broken | two senses: "fairly" OR "totally" — tone decides |
| | | |
| **ziemlich** | fairly / pretty | |
| Es ist ziemlich spät. | It's pretty late. | |
| | | |
| **kaum** | hardly | |
| Ich kann es kaum glauben. | I can hardly believe it. | |
| | | |
| **fast** | almost | |
| Ich bin fast fertig. | I'm almost done. | false friend: NOT English "fast" |
| | | |
| **besonders** | especially | |
| Es ist besonders kalt. | It's especially cold. | |
| | | |
| **etwas** | a bit / somewhat | |
| Es ist etwas teuer. | It's a bit expensive. | also = "something" (pronoun) |
| | | |
| **genug** | enough | |
| Das ist genug. | That's enough. | follows the word: "groß genug" = big enough |
| | | |
| **völlig** | completely / totally | |
| Das ist völlig falsch. | That's completely wrong. | |
| | | |
| **wirklich** | really | |
| Es ist wirklich gut. | It's really good. | |
