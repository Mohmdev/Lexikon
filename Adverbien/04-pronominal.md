# Pronominal adverbs — fusing "preposition + it"

## The everyday situation this solves

In English, when you talk about a thing you already mentioned, you say a **preposition + "it"**:

- I write **with it**.
- I'm waiting **for it**.
- We talked **about it**.

That's two words: a preposition (with / for / about) + the word **"it"**. Simple.

## German handles this differently — for THINGS, it glues them into one word

German does **not** say "with it" as two words for a thing. It fuses them into a single word:

- the glue piece **da-** means roughly **"it / that"**
- you stick the preposition onto it:  **da** + **mit** = **damit** = "with it"

So:

| English | German (fused) |
|---|---|
| with it | **damit** |
| for it | **dafür** |
| about it | **darüber** |
| on it | **darauf** |
| from it | **davon** |

⚠️ Notice the order flips vs English: English = *with + it*, German = *it(da) + with(mit)* = **damit**.
The preposition goes on the **end**.

(The fancy name "pronominal adverb" just means: a da-/wo- word standing in for a thing. Don't sweat
the label.)

## See it in real sentences

- Das Messer ist scharf. Ich schneide **damit**. — The knife is sharp. I cut **with it**.
- Der Stuhl ist alt. Ich sitze **darauf**. — The chair is old. I sit **on it**.
- Das Problem ist groß. Wir sprechen **darüber**. — The problem is big. We talk **about it**.

Each time, "damit / darauf / darüber" replaces "with/on/about + the thing just mentioned."

## The question version: swap da- ("it") for wo- ("what")

Use the **exact same trick**, but start with **wo-** (= "what") instead of **da-** (= "it"). Now it's
a question:

| statement (da- = it) | question (wo- = what) |
|---|---|
| **damit** = with it | **womit** = with what? |
| **dafür** = for it | **wofür** = for what? |
| **darüber** = about it | **worüber** = about what? |

- **Womit** schneidest du? — **With what** are you cutting?
- **Worüber** sprecht ihr? — **What** are you talking **about**?

## The little -r- (just for pronunciation)

When the preposition starts with a **vowel** (auf, an, in, über), German jams an extra **r** in the
middle so it's not two vowels crashing together:

| | da- form | wo- form |
|---|---|---|
| auf | da**r**auf (not "daauf") | wo**r**auf |
| an | da**r**an | wo**r**an |
| über | da**r**über | wo**r**über |
| in | da**r**in | wo**r**in |

That's all the -r- is — a pronunciation bumper. No deeper meaning.

## The ONE rule for when to use this

This fusion is **only for things and ideas**. For **people**, you do NOT fuse — you keep the normal
two-word form (preposition + him/her/them):

| | a thing | a person |
|---|---|---|
| with | **damit** (with it) | **mit ihm** (with him) |
| about | **darüber** (about it) | **über sie** (about her) |
| for | **dafür** (for it) | **für ihn** (for him) |

So the deciding question is simple: **thing → fuse it (damit). person → don't fuse (mit ihm).**

## Recap (the whole thing in 4 lines)

- **da-** + preposition = "preposition + **it**"  (statement: damit = with it)
- **wo-** + preposition = "preposition + **what**"  (question: womit = with what?)
- add **-r-** if the preposition starts with a vowel  (darauf, worauf)
- only for **things**, never people
