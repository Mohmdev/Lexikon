# Lokaladverbien (place adverbs)

*📍 Answer **wo? / wohin?** (where / where to).*

| German | English | Note |
|---|---|---|
| **hier** | here | |
| Ich wohne hier. | I live here. | |
| | | |
| **dort** | there | |
| Dort ist die Bank. | The bank is there. | |
| | | |
| **da** | there / here | |
| Ich bin da. | I'm here. | |
| da = also "in that case / then" | | da has a second, non-place sense (since/then) |
| | | |
| **oben** | above / upstairs | |
| Das Buch liegt oben. | The book is upstairs. | |
| | | |
| **unten** | below / downstairs | |
| Ich warte unten. | I'll wait downstairs. | |
| | | |
| **links** | (on the) left | |
| Das Haus ist links. | The house is on the left. | |
| | | |
| **rechts** | (on the) right | |
| Geh rechts. | Go right. | |
| | | |
| **überall** | everywhere | |
| Überall ist Schnee. | There's snow everywhere. | |
| | | |
| **nirgends** | nowhere | |
| Er ist nirgends. | He's nowhere. | also: nirgendwo |
| | | |
| **irgendwo** | somewhere | |
| Es muss irgendwo sein. | It must be somewhere. | |
| | | |
| **draußen** | outside | |
| Die Kinder spielen draußen. | The kids play outside. | |
| | | |
| **drinnen** | inside | |
| Bleib drinnen. | Stay inside. | |
| | | |
| **vorne** | at the front | |
| Ich sitze vorne. | I sit at the front. | |
| | | |
| **hinten** | at the back | |
| Das Gepäck ist hinten. | The luggage is in the back. | |
| | | |
| **weg** | away / gone | |
| Er ist weg. | He's gone. | |
| | | |
| **hin** | (to) there — away from speaker | |
| Wo gehst du hin? | Where are you going? | direction AWAY from the speaker |
| | | |
| **her** | (to) here — toward speaker | |
| Komm her! | Come here! | direction TOWARD the speaker |
