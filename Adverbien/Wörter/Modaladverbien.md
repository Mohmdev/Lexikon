# Modaladverbien (manner adverbs)

Answer **wie?** (how). This file covers only the pure adverbs, not adjectives.

Most manner words are **adjectives** used as adverbs (schnell, gut, leise); those live in
`Adjektive/`, not here. This file holds only the true adverbs, which cannot stand in front of a
noun.

| Deutsch | Anmerkung | Englisch |
|---|---|---|
| **gern** (gerne)| | gladly / like to |
| Ich trinke gern Kaffee. | = how you say "like doing" sth | I like drinking coffee. |
| gern → lieber → am liebsten | IRREGULAR; "lieber" = "I'd rather" | gladly → rather → most of all |
| **leider**| | unfortunately |
| Ich kann leider nicht kommen. | comments on the whole sentence | Unfortunately I can't come. |
| **anders**| | differently |
| Ich sehe das anders. | adverb of the adjective "andere" (other) | I see it differently. |
| **so**| | so / like this |
| Mach es so. | | Do it like this. |
| **vielleicht**| | maybe / perhaps |
| Vielleicht kommt er. | comments on the whole sentence | Maybe he'll come. |
| **hoffentlich**| | hopefully |
| Hoffentlich regnet es nicht. | speaker's wish about the sentence | Hopefully it won't rain. |
| **sowieso**| | anyway / in any case |
| Ich gehe sowieso. | | I'm going anyway. |
| **genauso**| | just the same / exactly so |
| Mach es genauso. | | Do it exactly the same way. |
