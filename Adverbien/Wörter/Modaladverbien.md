# Modaladverbien (manner adverbs)

*⚙️ Answer **wie?** (how) — only the pure adverbs that are NOT adjectives.*

Most manner words are just **adjectives** used as adverbs (schnell, gut, leise); those live in
`Adjektive/`, not here. This file holds only the real adverbs that are NOT adjectives — you can't
place them in front of a noun.

| German | English | Note |
|---|---|---|
| **gern** (gerne) | gladly / like to | |
| Ich trinke gern Kaffee. | I like drinking coffee. | = how you say "like doing" sth |
| gern → lieber → am liebsten | gladly → rather → most of all | IRREGULAR; "lieber" = "I'd rather" |
| | | |
| **leider** | unfortunately | |
| Ich kann leider nicht kommen. | Unfortunately I can't come. | comments on the whole sentence |
| | | |
| **anders** | differently | |
| Ich sehe das anders. | I see it differently. | adverb of the adjective "andere" (other) |
| | | |
| **so** | so / like this | |
| Mach es so. | Do it like this. | |
| | | |
| **vielleicht** | maybe / perhaps | |
| Vielleicht kommt er. | Maybe he'll come. | comments on the whole sentence |
| | | |
| **hoffentlich** | hopefully | |
| Hoffentlich regnet es nicht. | Hopefully it won't rain. | speaker's wish about the sentence |
| | | |
| **sowieso** | anyway / in any case | |
| Ich gehe sowieso. | I'm going anyway. | |
| | | |
| **genauso** | just the same / exactly so | |
| Mach es genauso. | Do it exactly the same way. | |
