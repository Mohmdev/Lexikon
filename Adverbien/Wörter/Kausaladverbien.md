# Kausaladverbien (cause adverbs)

Answer **warum?** (why).

Most of these also **connect clauses** and push the verb to slot 2 (konjunktional). In the
examples the verb stands directly after the adverb.

| Deutsch | Anmerkung | Englisch |
|---|---|---|
| **deshalb**| ⛓️ | therefore / that's why |
| Es regnet, deshalb bleibe ich zu Hause. | verb "bleibe" jumps to slot 2 after deshalb | It's raining, so I'm staying home. |
| **deswegen**| ⛓️ | for that reason |
| Ich bin krank, deswegen komme ich nicht. | | I'm sick, that's why I'm not coming. |
| **darum**| ⛓️ | that's why |
| Darum bin ich hier. | | That's why I'm here. |
| **trotzdem**| ⛓️ | nevertheless / anyway |
| Es regnet. Trotzdem gehe ich raus. | | It's raining. I'm going out anyway. |
| **also**| | so / therefore |
| Du bist müde, also schlaf. | false friend: NOT English "also" | You're tired, so sleep. |
| **daher**| ⛓️ | therefore / hence |
| Es ist spät, daher gehe ich. | | It's late, hence I'm leaving. |
| **dennoch**| ⛓️ | nevertheless |
| Es war schwer, dennoch schaffte er es. | contrast | It was hard, nevertheless he managed. |
| **jedoch**| ⛓️ | however |
| Ich wollte kommen, jedoch hatte ich keine Zeit. | contrast | I wanted to come, however I had no time. |
| **außerdem**| ⛓️ | besides / moreover |
| Es ist teuer; außerdem ist es hässlich. | addition | It's expensive; moreover it's ugly. |
| **sonst**| ⛓️ | otherwise / or else |
| Beeil dich, sonst kommen wir zu spät. | | Hurry, otherwise we'll be late. |
| **allerdings**| ⛓️ | however / though |
| Es ist schön, allerdings teuer. | | It's nice, though expensive. |
| **nämlich**| | you see / because |
| Ich gehe; ich bin nämlich müde. | gives a reason; never starts the sentence | I'm leaving; you see, I'm tired. |
| **zwar**| | admittedly / it's true |
| Es ist zwar kalt, aber schön. | usually paired: "zwar … aber" | It's cold, admittedly, but nice. |
| **folglich**| ⛓️ | consequently |
| Er war krank, folglich blieb er zu Hause. | formal | He was sick, so he stayed home. |
