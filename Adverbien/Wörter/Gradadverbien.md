# Gradadverbien (degree adverbs)

*🔆 Answer **wie sehr?** (how much).*

These don't take a sentence slot; they **cling to an adjective or adverb** and crank its intensity.

| Deutsch                   | Anmerkung                                                     | Englisch                       |
| ------------------------- | ------------------------------------------------------------- | ------------------------------ |
| **sehr**                  |                                                               | very                           |
| Es ist sehr kalt.         | intensifies adj/adv only; verbs use "viel" (ich arbeite viel) | It is very cold.               |
| **zu**                    |                                                               | too (excessively)              |
| Das ist zu teuer.         | careful: "zu" is also the preposition "to"                    | That's too expensive.          |
| **ganz**                  |                                                               | quite / completely             |
| ganz gut / ganz kaputt    | two senses: "fairly" OR "totally" — tone decides              | quite good / completely broken |
| **ziemlich**              |                                                               | fairly / pretty                |
| Es ist ziemlich spät.     |                                                               | It's pretty late.              |
| **kaum**                  |                                                               | hardly                         |
| Ich kann es kaum glauben. |                                                               | I can hardly believe it.       |
| **fast**                  |                                                               | almost                         |
| Ich bin fast fertig.      | false friend: NOT English "fast"                              | I'm almost done.               |
| **besonders**             |                                                               | especially                     |
| Es ist besonders kalt.    |                                                               | It's especially cold.          |
| **etwas**                 |                                                               | a bit / somewhat               |
| Es ist etwas teuer.       | also = "something" (pronoun)                                  | It's a bit expensive.          |
| **genug**                 |                                                               | enough                         |
| Das ist genug.            | follows the word: "groß genug" = big enough                   | That's enough.                 |
| **völlig**                |                                                               | completely / totally           |
| Das ist völlig falsch.    |                                                               | That's completely wrong.       |
| **wirklich**              |                                                               | really                         |
| Es ist wirklich gut.      |                                                               | It's really good.              |
