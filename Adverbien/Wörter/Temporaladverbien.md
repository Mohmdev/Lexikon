# Temporaladverbien (time adverbs)

*⏰ Answer **wann?** (when).*

| Deutsch                          | Anmerkung                                  | Englisch                              |
| -------------------------------- | ------------------------------------------ | ------------------------------------- |
| **heute**                        |                                            | today                                 |
| Heute ist Montag.                |                                            | Today is Monday.                      |
| **morgen**                       |                                            | tomorrow                              |
| Ich komme morgen.                |                                            | I'm coming tomorrow.                  |
| morgen vs **der** Morgen         | same word — capital M = the noun "morning" | tomorrow vs "the morning"             |
| **gestern**                      |                                            | yesterday                             |
| Gestern war es kalt.             |                                            | Yesterday it was cold.                |
| **jetzt**                        |                                            | now                                   |
| Ich gehe jetzt.                  |                                            | I'm going now.                        |
| **oft**                          |                                            | often                                 |
| Ich gehe oft ins Kino.           |                                            | I often go to the cinema.             |
| oft → öfter                      | umlaut in comparative                      | often → more often                    |
| **immer**                        |                                            | always                                |
| Sie ist immer pünktlich.         |                                            | She is always on time.                |
| **nie**                          |                                            | never                                 |
| Ich trinke nie Kaffee.           |                                            | I never drink coffee.                 |
| **dann**                         | ⛓️                                         | then (next / in that case)            |
| Erst esse ich, dann schlafe ich. | sequence — NOT "when"                      | First I eat, then I sleep.            |
| **danach**                       | ⛓️                                         | afterwards                            |
| Wir essen, danach gehen wir.     |                                            | We eat, afterwards we leave.          |
| **zuerst**                       | ⛓️                                         | first / at first                      |
| Zuerst kommt die Arbeit.         |                                            | First comes work.                     |
| **schließlich**                  | ⛓️                                         | finally / eventually                  |
| Schließlich kam er doch.         |                                            | He finally came after all.            |
| **inzwischen**                   | ⛓️                                         | meanwhile                             |
| Inzwischen wurde es dunkel.      |                                            | Meanwhile it got dark.                |
| **bald**                         |                                            | soon                                  |
| Ich komme bald.                  |                                            | I'm coming soon.                      |
| **schon**                        |                                            | already                               |
| Bist du schon hier?              |                                            | Are you already here?                 |
| **noch**                         |                                            | still / yet                           |
| Ich bin noch hier.               | "noch nicht" = not yet                     | I'm still here.                       |
| **wieder**                       |                                            | again                                 |
| Er ist wieder da.                |                                            | He's back again.                      |
| **manchmal**                     |                                            | sometimes                             |
| Manchmal regnet es.              |                                            | Sometimes it rains.                   |
| **selten**                       |                                            | rarely / seldom                       |
| Ich bin selten krank.            |                                            | I'm rarely sick.                      |
| **gerade**                       |                                            | just now / right now                  |
| Ich esse gerade.                 | also = "straight" (adj)                    | I'm eating right now.                 |
| **früher**                       |                                            | earlier / in the past                 |
| Früher war alles anders.         | comparative of "früh"                      | In the past everything was different. |
